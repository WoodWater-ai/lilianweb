# 礼品兑换系统官网 - React项目

## 项目简介

这是礼享科技礼品兑换系统的官方网站，采用React + TypeScript + Tailwind CSS构建，展示了行业首个Agent智能解决方案。

## 技术栈

- **React 19** - 前端框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架

## 项目结构

```
official-website/
├── src/
│   ├── components/
│   │   ├── Navbar.tsx          # 导航栏组件
│   │   ├── Hero.tsx            # 英雄区域组件
│   │   └── AgentSection.tsx    # Agent特色区域组件
│   ├── App.tsx                 # 主应用组件
│   ├── main.tsx                # 应用入口
│   └── index.css               # 全局样式
├── index.html                  # HTML模板
├── tailwind.config.js          # Tailwind配置
├── postcss.config.js           # PostCSS配置
└── package.json                # 项目依赖
```

## 核心功能

### 1. Agent智能解决方案
- 自然语言交互
- 自动化业务流程
- 智能数据分析
- 精准执行（99.2%准确率）

### 2. 产品特色
- 完整业务闭环
- 多租户架构
- 移动端优先
- 快速部署
- 安全可靠

### 3. 系统架构
- 管理后台：商品、订单、分类、礼包、卡密管理
- 移动商城：双登录、智能兑换、订单跟踪

## 开发命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```
